#  React + Django Full Stack APIs

This repository contains full stack web development projects using **React.js (Frontend)** and **Django REST Framework (Backend)** with MySQL database integration.

---

## 🔹 Tech Stack

### Frontend:
- React.js
- JavaScript (ES6+)
- HTML, CSS
- Axios (API calls)

### Backend:
- Django
- Django REST Framework (DRF)
- MySQL

---

## 🔹 Features

- REST API Development
- CRUD Operations
- API Integration with React
- Scalable Backend Architecture
